# Quick Installation Guide

## Chrome Installation
1. Go to `chrome://extensions/`
2. Enable "Developer mode"
3. Click "Load unpacked"
4. Select the `unified-extension` folder
5. Pin the extension from the Chrome toolbar

## Safari Installation
1. Open Safari > Settings > Extensions
2. Enable "Allow unsigned extensions"
3. Click "Add Extension..."
4. Select the `unified-extension` folder
5. Enable the extension and grant YouTube permissions

## Usage
- Visit YouTube Shorts to start tracking
- Click the extension icon to view stats and configure limits
- Get notifications when you reach your limit
